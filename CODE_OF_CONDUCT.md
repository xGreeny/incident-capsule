# Code of conduct

Participation in this project requires professional, respectful, and technically constructive communication.

Contributors are expected to:

- discuss code and evidence-handling decisions without personal attacks;
- avoid publishing sensitive operational data or real incident material;
- give clear technical reasoning when proposing broader collection scope;
- accept maintainer decisions made to protect safety, privacy, or project focus;
- report security concerns privately.

Harassment, discriminatory conduct, deliberate disclosure of sensitive data, and repeated bad-faith disruption are not accepted. Maintainers may edit or remove content and restrict participation when needed to protect contributors and users.
