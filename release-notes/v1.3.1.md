## AppCompatCache export fix

Incident Capsule 1.3.1 is a maintenance release that fixes the raw AppCompatCache (Shimcache) export in the ExecutionArtifacts collector.

### Highlights

- The AppCompatCache REG_BINARY value was enumerated to `object[]` before the collector's byte-array type check, so `appcompatcache.bin` was never written and `AppCompatCacheBytes` reported `0` on every host. The value is now normalized and exported (verified end to end: a 367 KB Shimcache blob with a stable SHA-256).
- The precise AppCompatCache warning added in 1.2.1 is what surfaced this defect from a field capsule.

Update recommended for anyone relying on Shimcache evidence from the ExecutionArtifacts collector.
